本周一直在倒腾 Obsidian+ai 做笔记的事，最后落在了 para+zk 卡片笔记的组合上。
在这之前，退订了 Gemini 的网页版，开始使用 api 的方式接入。也同时购买了 ds 的 api，混合使用。这个过程中有很多麻烦事，主要都是集中在解决 g 的 api 不稳定，回过头看都是 cloudflare 那一侧 work 脚本的问题。没办法，谁叫咱只能 vibe coding，不懂就只能一个一个问题碰。但毕竟咱是实用主义，配好了就行。

做笔记这个事情还是挺让人头疼的，因为有了 ai，我感觉自己的每天遇到的信息都值得被好好对待了，于是认真研究这些理论和如何实践。现在笔记体系大致成型了，还需要优化。在这部分建架构花费的精力不少，除了如何设计文件夹层级，还要反复调整我那几个关键的提示词模板和文章模板，其严谨程度这次不亚于工作。

好玩的是，一旦调整好，我反而更有兴趣把过去的一些似懂非懂的知识梳理一遍了，特别像动物的反刍过程。说实话，这块还挺费时间的。现在貌似更有动力去获取一些高价值的信息，并入我的知识体系里。我还没能参透这套方法的精髓和益处，但本身我也需要记录，也有了比较合理的方式，坚持下来，慢慢应该会有变化。

儿子这周放假第一天，明显很亢奋。说到了八字的问题，他说他缺木，我说我正好就是木，然后和他解释了几句我也不太懂的八字的那些东西，他大概听懂了他来找我吃木头，然后就假装啃我的床去了。下周父母来，带娃 10 天。买了单人帐篷，以防万一有人被踢去客厅睡。

工作上，把 3 个专利搞差不多了，提了 2 个，还差1 个。写八股文现在真快啊。
