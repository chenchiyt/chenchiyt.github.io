今晚异常烦躁

2010-07-27 3 min read # 杂谈

脖子疼。晚上人其实已经来的差不多了，但是还是真正的忙了一晚上。

都是些什么事儿呢？今天有些人问的问题特别的多，比如怎么上不去网啊，你能给我订南京的房间么，我想订什么时候的房间，啊，我再考虑考虑吧，你好啊，还是我，哦这样啊，那我再打过来吧，哈喽，又是我……

还有： 上不去网啊？上不去网啊？怎么上网啊？怎么怎么上网啊？？？

估计是阿姨放拖鞋的时候忘记了锁108房间的门，这个美国客人找到我这里来了，一句excuse
me你来一下把我叫去了，我简直是给人唱了一首sorrysorry才回来。

随后，又有如下问题：哪儿有开水啊，开水在哪儿啊，开水开水开水啊，外加各种方言版。

非常吵，一帮广东孩子在院子里高声喧闹，不停闹。一个一个人过来买饮料。

这一些都是从吃饭吃了一半开始的。有几个学生要我们给订南京的旅社。景芳出去有事，临走前还特地说了句你们让他吃了再订。要是没有这句话，我估计连饭都吃不完。真的很多人就有这样的想法，我在你这里住，给你钱了，我就是爷，不管你手头多忙有几个人，我来了你就是得处理我的事情。又想起来那天几个客人一起在前台站着，同时向我提问题提要求的时候了。

今晚真阴暗。

说点好的

Isabel看过我的照片，说里面的确有些很不错，说可以考虑出书，或明信片这些。我很高兴能有人看懂我的照片，很多人看的时候都太随意，甚至都不知道重点是什么。

晚上呢，又挨了点冷水，同志们说这个难度太大，即使是有才能，想给明信片供稿赚钱或者出书还是难的。先继续积累吧，等出国了的，来一个一发不可收拾，国外就不大会有印刷明信片的偷照片来印的现象了。

artistic side, and logical side, I want to develope both, but I think now that
I need a priority, which seems more likely the latter one.
