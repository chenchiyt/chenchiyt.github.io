虎年到

2010-02-14 2 min read # 400d # 机身 # 佳能镜头 # 焦段 # 红圈 # 杂谈

虎年到了，同时21世纪的第一个10年已经过去了。

十年的变化时巨大的，所以我有充足的理由去期待下一个十年的到来，人们的生活将会发生巨大的变化，而我，将成为这一切的见证人；当然，我也会努力成为弄潮儿中的一份子。

窗外爆竹的巨响对我来说已经没有了任何的吸引力，原来我长大了，真的大了。

现在张口闭口想的全都是自己的前途问题。

前途或为天堂，或为炼狱。我想，我是不会选择一条普普通通的路来走完我的一生的。

未知，困难，考验。通通的来吧，我准备好了。

在摄影的道路上，我也会向前迈进一步，更加成熟，更加有深度。今天和爸爸一个同学聊了聊，那个叔叔搞了一辈子摄影了，他推荐我现在将400D的机身换成7D，说这样会比投资70-200
F4划算。，整体性能上有更大的提高。我做了一下比较，的确有道理。

400D在很多功能上有些太鸡肋了，虽说我觉得还不错，但是要真正的专业机身，400D算不上的。

70-200的焦段，之前一直想用来捕捉人们最自然的神情，还有身边环境的细节的，但是想了想，100-200这部分的使用频率所带来的变化不会比将我现有的镜头的表现同时升级要来的实在。

换机身这点的确有道理。但是推荐18-135这点我就不是很赞同了，如果说这个头的成像要比同等价位的佳能镜头要好不少我可以接受，但是要说这个直逼红圈，那我还是怀疑一下，佳能怎么可能会突然变得这么够意思。想到了我最原来那支28-105，我就知道了这样的大变焦不可靠。
