感慨于又一年的工作热潮

2010-10-03 4 min read # 宝洁 # 玛氏公司 # 逻辑能力 # 英语 # 动物医学 # 出国 # 教育

我在开始申请出国之前，就非常清楚：读书不是为了混一个学历，混一个面子，混一分人模狗样但没有含金量的职业，而是真正的学到能应用到工作中的东西，能够在别人束手无策的时候，我能响亮的打出三张A，一个炸弹，茅塞顿开。

对我来说，这才是读书的目的所在，为更好的职业发展打下理论基础，同时还不能抛弃实践的考验。对我来说，任何纯理论性的教育都是失败的，同时，不足称的实习课也没有资格被称作实践。

在之前家里人学校人父母同事们的潜移默化的影响下，我一直走的就是出国深造的路线。考英语，做试验。然后呢？
下文其实断了，因为在这一切发生之前我根本没有想好，自己究竟看得起什么，看不起什么。这不是我装专家指点江山，而是很主观的从我内心想要的生活状态来判断的，是不需要外人评论的特质。

然后我就想，我到底要什么呢？

我要的是保持思维的活跃，一个鼓励创新的思想环境，一个不安逸于是能让人不停完善自己向上走的人际圈子。

于是就说到了我为什么又悸动了。虽然认为自己已经考虑好了这个问题，但是今天发现还是有不缜密的地方，于是还是不爽了一会儿。

网上多个外企发布了招聘信息，于是宝洁啊，玛氏公司这些牛企纷纷打出了旗号，宣布招人。同学说，去试试吧，你英语多好啊？于是我看了看宝洁。 牛！
然后我又看了很多人分享的经验，发现自己弱势有二：一，专业背景；二，工作经验。
和独木桥上的数千人相比，我不认为我动物医学的背景能给我加分，反而是扣分；实验室两年，几篇SCI挂名，即使第一作者能说明什么？好吧，姑且算自己有领导力，很强的逻辑能力，但是和在实习公司混迹两年有了等同于我的成就的同学比，企业更看重哪个呢？当然是看重和自己近的这个。
于是我又看了看玛氏。 牛！ 除了好吃的德芙之外，他还经营宠物用品，这不就挺对口了么？
但是，看看他们网申的题是什么？写项目策划书。没有任何经验的我又OUT了：不是不敢尝试，而是说机会不大，投入的时间会损坏其他机会。

所以分析来分析去，如果找工作，我的背景里面只有英语能力可是说是一个人家认可的刚性指标，所以在这方面，我是瘸腿的。
有瘸腿的一方面，于是我很不爽：我对自己的要求向来是只要知道的东西，掌握程度应该和需要程度成正比，就好比我之前打算出国，那么英语就一定要做到我群体中最好的程度。

刚才说过，我想要的那些状态，公司是可以给我的，我发现我在需要这个能力的时候没有在我的口袋里找到。

还有什么地方能够给我那种状态么？如果我选择继续科研，那么顶级的大学应该可以，我相信世界上最聪明的人凑在一起应该是很好玩儿的事情。但是，那样我就基本得努力留在国外，但是我还是希望毕业几年后可以在国内的，这里有我的家人，朋友，再怎么说，这是自己的国家。

于是只能这样做：现在无力工作（我指适合我期望状态的工作），只能选择出国。出国后，在读书期间充分利用假期补回所有找工作需要的经验，理论，还有能力，在毕业的时候能够将学校习得的理论，实习获得的感触结合起来，以一个经过博士训练的实践者的姿态去进入职场。或许如果命运眷顾，我也还会有其他的选择。

我想，如果不想平庸的死去，这应该是能看得清楚的一部分正道。
